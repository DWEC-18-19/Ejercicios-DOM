Añadir otro botón arriba con clase "up" a cada elemento li de esta forma:   

<li>grapes 
          <button class="up">Arriba</button>          
          <button class="remove">Eliminar</button>
</li>

Intercambiar el elemento superior por el pulsado para que suba un nivel. 
Pista, usar la propiedad previousElementSibling del elemento li pulsado y el método insertBefore del elemento ul para intercambiarlos.
